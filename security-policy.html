Root Work Framework

Security Policy

Root Work Framework takes the security of our users' data and our systems seriously.

Reporting a Vulnerability

If you discover a security vulnerability, please disclose it responsibly by emailing us at hearn.sa@gmail.com.

We will respond within 5 business days to acknowledge your report and outline next steps.

Please include the following in your report:

  • A clear description of the vulnerability
  • Steps to reproduce the issue
  • The potential impact of the vulnerability

We credit researchers who follow responsible disclosure practices in our acknowledgements.

Scope

In scope:

  • The main site: fg2g-rwfw.com
  • API endpoints: /api/chat, /api/newsletter, /api/consultation, /api/unsubscribe

Out of scope:

  • Third-party services we rely on, including Vercel infrastructure, n8n, and the Anthropic API

Please report vulnerabilities in third-party services directly to those providers.

Safe Harbor

We will not pursue legal action against researchers who discover and report security vulnerabilities in good faith and in accordance with this policy.

We ask that you:

  • Do not perform denial-of-service (DoS) testing
  • Do not access, modify, or delete user data beyond the minimum necessary to demonstrate the vulnerability
